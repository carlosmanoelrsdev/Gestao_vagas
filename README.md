# API Gestao-Vagas

Projeto desenvolvido para estudo de API Restful, com foco no gerenciamento de vagas, candidatos e companhias.

## Sobre o projeto

A API Gestao Vagas tem como objetivo permitir o gerenciamento de vagas e o acesso de candidatos e companhias ao sistema.

O projeto está sendo desenvolvido com foco em boas práticas de APIs REST, segurança e organização de código.

## Funcionalidades

- Cadastro e gerenciamento de vagas
- Cadastro e envio de currículo
- Login seguro no sistema
- Rotas protegidas com Spring Security
- Autenticação com JWT
- Estrutura em desenvolvimento

## Tecnologias utilizadas

- Java
- Spring Boot
- Spring Security
- JWT
- Maven
- Docker
- PostgreSQL

## Estrutura do projeto

O projeto está organizado seguindo a estrutura MVC, com separação entre:

- Controllers
- DTOs
- Entities
- Security
